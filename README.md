# ZenlessGrid – Public Issue Tracker
---

## What is ZenlessGrid?

**ZenlessGrid** is a website built for players of **Zenless Zone Zero (ZZZ)**. It fetches player profile data and presents it in a neat, easy-to-read, and shareable format.

The main ZenlessGrid codebase is **private**, but this repository exists so anyone can:

* Report bugs or broken behavior
* Suggest new features or improvements
* Share UI/UX ideas
* Give general feedback

---

## A Few Things to Know

* This repo is **only for issues and feedback**
* Pull requests are **not accepted**

You can check out the live site here:
 **[zenlessgrid.zone](https://zenlessgrid.zone)**

---

## Bug Reports

Found something that doesn’t look right or isn’t working?

When opening a bug report, try to include:

* What you expected to happen
* What actually happened
* Steps to reproduce the issue (if possible)
* Screenshots or screen recordings
* Your browser and device

---

## Feature Requests & Ideas

Got an idea that would make ZenlessGrid cooler or more useful?

Open an issue and let me know:

* What the feature is
* Why you think it’d be helpful
* Any references, mockups, or examples (optional but appreciated)

Even small ideas are welcome!

---

## Labels You’ll See

Issues may be tagged to keep things organized:

* `bug` – Something’s broken
* `enhancement` – Improving an existing feature
* `feature-request` – New feature ideas
* `ui/ux` – Visual or usability feedback
* `question` – General questions

---

## Feedback & Community

ZenlessGrid is built for the ZZZ community, and feedback really helps shape the site. If something feels off or you think something could be better, don’t hesitate to open an issue.

Thanks for helping improve ZenlessGrid

---

## Data

ZenlessGrid is powered by the [Enka.Network](https://enka.network/), which provides the underlying player profile data used on the site.
